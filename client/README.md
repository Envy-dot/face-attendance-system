# Face Attendance System - Client

A React + Vite web application for facial recognition-based attendance tracking.

## Features

- Real-time face detection and recognition
- Attendance logging and management
- User-friendly interface
- Fast development with Vite HMR

## Tech Stack

- **React** - UI library
- **Vite** - Build tool and dev server
- **ESLint** - Code quality
- **Face Detection API** - Facial recognition

## Getting Started

### Prerequisites

- Node.js 16+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Runs the app with Vite HMR enabled.

### Build

```bash
npm run build
```
